Ürün Sayfası

VidExtract

Python tabanlı video/ses ayrıştırma ve indirme aracı. Bu sayfa; özellikler, demo arayüz ve web'e taşıma seçeneklerini gösterir.

Hızlı

CLI veya GUI ile hızlı akış.

Kontrol

Format, kalite, ses/video ayrımı.

Güvenli

Kendi makinenizde/servisinizde çalıştırma.

Web Demo (Statik)

Bu form sadece arayüz demodur. Statik hosting (GitHub Pages vb.) Python çalıştıramaz. Gerçek indirme/ayrıştırma için backend gerekir.

Peki bunu webde gerçekten çalıştırabilir miyim?

1) Backend + Web UI (Önerilen)

VidExtract'i bir API'ye koyarsın (FastAPI/Flask). Web arayüzü bu API'ye link gönderir, sunucu indirir ve dosyayı kullanıcıya verir.

2) Masaüstü (Electron/Tauri)

Web gibi görünen arayüz + arkada Python çalıştırma. En stabil seçeneklerden.

3) Tarayıcı içi Python (Sınırlı)

Pyodide gibi çözümlerle bazı Python kodları tarayıcıda çalışır; fakat video indirme/ffmpeg gibi şeyler genelde kısıtlıdır.

Özet

Evet, çoklu sayfa site yapıp VidExtract için ayrı bir sayfa hazırlayabilirsin. Ama gerçek çalıştırma için en mantıklısı: web arayüz + backend API.
